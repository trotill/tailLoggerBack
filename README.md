## Description

Log file reader over WEB (backend).     
Reading a file is similar to the principles of the tail -f -n xxx utility

## NodeJs version
NodeJs v20.15.1

## Installation

```bash
$ yarn install
```

## Running the app

```bash
$ yarn start
```
open URL http://localhost:6200

swagger api/docs http://localhost:6200/api/docs
